Consultorio DR. Amado Grupo7 1
Gestión de Turnos en Consultorios es una aplicación diseñada para facilitar la programación y gestión de citas en consultorios médicos. Este software permite a los administradores registrar pacientes, programar, modificar y eliminar turnos de manera eficiente. Con una interfaz fácil de usar, los administradores pueden visualizar los turnos en un DataGridView y filtrar por criterios específicos como DNI. Desarrollado en Visual Studio y utilizando MySQL como base de datos, esta aplicación mejora la experiencia del paciente y optimiza la administración de citas en el consultorio.

Descripción detallada

Clase Form2 que hereda de Form. Esta clase es responsable de manejar la interfaz de usuario para la gestión de turnos en el consultorio.

Variables:

  • connectionString: Cadena de conexión para la base de datos.
  • idPaciente: Variable entera que almacena el ID del paciente seleccionado para operaciones de modificación y eliminación.
  • isLoading: Bandera booleana que indica si los datos están siendo cargados en el DataGridView para prevenir eventos no deseados durante la carga.
