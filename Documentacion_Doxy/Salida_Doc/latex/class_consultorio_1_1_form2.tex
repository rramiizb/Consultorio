\doxysubsection{Descripción detallada}
Clase \doxylink{class_consultorio_1_1_form2}{Form2} que hereda de Form. Esta clase es responsable de manejar la interfaz de usuario para la gestión de turnos en el consultorio.

Variables\+:
\begin{DoxyItemize}
\item connection\+String\+: Cadena de conexión para la base de datos.
\item id\+Paciente\+: Variable entera que almacena el ID del paciente seleccionado para operaciones de modificación y eliminación.
\item is\+Loading\+: Bandera booleana que indica si los datos están siendo cargados en el Data\+Grid\+View para prevenir eventos no deseados durante la carga. 
\end{DoxyItemize}
